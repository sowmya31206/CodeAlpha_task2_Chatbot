## FAQ Chatbot

This project was developed as Task 2 for the CodeAlpha Artificial Intelligence Internship 

## Project Description

The FAQ Chatbot is an AI-powered application that answers frequently asked questions by finding the most relevant response using Natural Language Processing (NLP)

## Features

- Answers frequently asked questions
- Uses Natural Language Processing (NLP)
- TF-IDF Vectorizer for text representation
- Cosine Similarity for finding the best match
- Interactive Gradio user interface
- Fast and easy to use

## Technologies Used

- Python
- NLTK
- Scikit-learn
- Gradio
- Google Colab

## Project Structure

```
FAQ_Chatbot.ipynb
README.md
requirements.txt
```

## How to Run

1. Install the required libraries.
2. Open the notebook in Google Colab.
3. Run all the cells.
4. Open the Gradio link.
5. Ask your questions and get instant answers.

## 📖 Sample Questions

- What is Artificial Intelligence?
- What is Machine Learning?
- What is Python?
- Will I receive a certificate?
- How can I contact support?

## Outcome

This project demonstrates how NLP techniques such as TF-IDF Vectorization and Cosine Similarity can be used to build a simple and effective FAQ Chatbot.
